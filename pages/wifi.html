WiFi Network

This page contains all the information to access the internet network during the ASAP 2018 conference.

Eduroam network

POLIMI is an eduroam member. If your univeristy is also a member, you can use directly that network.

If you have any problem with the eduroam network, visit this page [link].

Non Eduroam users

If you do not have access to eduroam network, please visit the registration desk and ask for a WiFi certificate. For the italian law, we need to register you with an unique Internet identifier. For this reason we need you to provide a valid ID for the WiFi account registration.
